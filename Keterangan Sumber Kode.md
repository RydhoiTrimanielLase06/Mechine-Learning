# Proyek Praktik Machine Learning - Belajar Machine Learning untuk Pemula

## Deskripsi
Repositori ini berisi kode hasil pengerjaan modul latihan/proyek akhir dari kursus **Belajar Machine Learning untuk Pemula** di platform **Dicoding Indonesia**.

Proyek ini mencakup implementasi:
1. Clustering.ipynb
2. Feature_engineering.ipynb
3. Klasifikasi.ipynb
4. Optimisasi_model_dengan_hyperparameter_tuning.ipynb
5. Overfitting_dan_underfitting.ipynb
6. Pemilihan_fitur.ipynb
7. Regression.ipynb 

## Sumber Materi & Atribusi
Kode dalam repositori ini dikembangkan berdasarkan materi pembelajaran dari:
* **Penyedia Materi**: [Dicoding Indonesia](https://www.dicoding.com/)
* **Tujuan**: Portofolio pribadi dan bukti penyelesaian modul pembelajaran.

## Lisensi
Kode asli adalah hak milik intelektual dari **Dicoding Indonesia**. Repositori ini dibuat hanya untuk tujuan pendidikan dan dokumentasi proses belajar pribadi.
